These files contain the commands to recreate the supplementary analyses for "Testing Islam's Political Advantage: Evidence from Indonesia." Note that these files do not create the figures and tables themselves, they only provide the results needed to create them.

readme.txt
	This file that you are reading right now.

table2.do - table8.do
	Generates the results for tables 2-8

figure 1-2.do
	Generates the results for figure 1 and figure 2
